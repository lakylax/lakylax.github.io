# Laxman Puri — Tech Portfolio

A clean, responsive single-page portfolio built using vanilla HTML5, CSS3, and modern JavaScript. Optimized for quick updates and fast load times on GitHub Pages.

## ⚡ Tech Stack & Performance
* Pure frontend architecture — no heavy framework dependencies or build pipelines.
* Dynamic DOM generation driven by a centralized data file (`js/data.js`).
* Interactive canvas backgrounds and layout animations out-of-the-box.
